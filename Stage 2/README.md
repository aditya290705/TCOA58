# Stage 2

## Campus Notification Frontend

This is a responsive React/Next.js frontend application for displaying campus notifications.

## Features

- Displays all notifications
- Displays priority notifications
- Filter by notification type
- Supports Event, Result and Placement notification types
- Supports pagination
- Supports limit query parameter
- Shows new and viewed notifications differently
- Uses Material UI for styling
- Runs on localhost:3000

## API Used

http://4.224.186.213/evaluation-service/notifications

## Query Parameters Used

- limit
- page
- notification_type

## Notification Types

- Event
- Result
- Placement
